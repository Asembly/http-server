# HTTP‑сервер на Java

### Возможности

- Выдача статики
- Маршрутизация запросов.
- Парсинг HTTP.
- Конфигурационный файл
- **Thread pool** соединений.
- **Reverse proxy**.
- Cache:
  - ***Кэширование на данный момент реализовано на базовом уровне. Создан обычный Lazy Cache который кэширует каждый запрос без обновления и удаления.***
- LoadBalancer: 
  - ***Сейчас реализован простой Round Robin алгоритм, в последующих апдейтах будет добавлен health-check.***

### В разработке

- Более сложная логика кэширования и проксирования с помощью LoadBalancer
- Поддержка **HTTPS (TLS‑шифрование)**.

Проект учебный, API и архитектура могут меняться по мере разработки.

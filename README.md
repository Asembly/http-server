# Простой HTTP‑сервер на Java

Минималистичный учебный HTTP‑сервер, написанный с нуля без сторонних фреймворков.

## Возможности

- Маршрутизация запросов через `addHandler()` (обработка разных роутов и методов).
- Парсинг HTTP‑заголовков и стартовой строки запроса.
- Поддержка разных `Content-Type`:
  - `application/json` (парсинг тела в JSON),
  - `multipart/form-data` (обработка форм и файлов).
- Модель «один поток — одно соединение».

## В разработке

- Работа в режиме **reverse proxy** (проброс запросов на другие сервисы).
- **load balancer** между несколькими бэкендами.
- Поддержка **HTTPS (TLS‑шифрование)**.

Проект учебный, API и архитектура могут меняться по мере разработки.
